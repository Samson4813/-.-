# Научиться Учиться
**Учебный проект по верстке с макета.**  
На странице можно просматривать видео, присутствуют анимации и активные ссылки.
Использованы технологии:
* flexbox
* анимации при помощи keyframes
* контейнер iframe
* пути к файлам организованы по БЭМ
